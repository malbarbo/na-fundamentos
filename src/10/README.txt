Você pode usar o comando

python3 gerador-de-pontos.py n > nome-arquivo

trocando n por um número é nome-arquivo por uma nome de um arquivo para
criar um arquivo com n pontos cartesianos com coordenadas entre 0 e 1.

Em seguida você pode usar o comando

python3 pontos-mais-distantes.txt nome-arquivo

para encontrar os pontos mais distantes que foram gerados.
